# Endless Runner Game (Unity)

## Project Overview
This project is an Endless Runner game developed in Unity. The goal of the game is to survive as long as possible while avoiding obstacles and collecting items.

## Game Concept
The player will control a character that continuously moves forward. The player must jump or avoid obstacles to stay alive. The difficulty will increase over time.

## Assets
Some assets may be used from free sources such as the Unity Asset Store or itch.io. Any assets used will be credited here.

## Development Plan (Next Two Weeks)

Week 1:
- Set up player movement (running and jumping)
- Create obstacles and basic level
- Add collision and game over system

Week 2:
- Add scoring system
- Add collectibles
- Improve UI (start screen and game over screen)
- Add sound and polish gameplay
